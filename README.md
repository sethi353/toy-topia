# ToyTopia - React Toy Store

[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

ToyTopia is a modern and interactive toy store web application built with **React**, **Firebase Authentication**, and **TailwindCSS**. It allows users to browse toys, view details, register/login, and enjoy a responsive and dynamic UI experience.

---

## ðŸŒŸ Features

- **User Authentication:** Login and Register using email/password or Google Sign-In (Firebase Auth).
- **Responsive Navbar:** Mobile-friendly navigation with dropdown menu.
- **All Toys Page:** Browse all toys with sorting and filtering options.
- **Toy Details Page:** View detailed information about each toy.
- **Sticky Navbar:** Navbar remains visible when scrolling.
- **Interactive UI:** Modern design with TailwindCSS components.
- **Promotions & Newsletter:** Sections for offers, promotions, and newsletter subscription.
- **Protected Routes:** Certain pages accessible only to authenticated users.
